# Mi Página Personal

Una página web personal moderna y profesional creada con HTML, CSS y JavaScript.

## 🚀 Características

- Diseño moderno con tema oscuro
- Totalmente responsive (se adapta a móviles y tablets)
- Animaciones suaves y efectos visuales
- Secciones: Inicio, Sobre Mí, Proyectos, Habilidades, Contacto
- Menú de navegación interactivo
- Formulario de contacto
- Enlaces a redes sociales

## 📁 Archivos

- `index.html` - Estructura principal de la página
- `styles.css` - Estilos y diseño
- `script.js` - Interactividad y animaciones

## 🌐 Cómo subir a GitHub Pages

1. Ve a https://github.com/new
2. Crea un repositorio llamado: `tu-usuario.github.io` (reemplaza "tu-usuario" con tu nombre de GitHub)
3. Marca como público
4. Sube los archivos: index.html, styles.css, script.js
5. Ve a Settings → Pages
6. Selecciona la rama "main" como fuente
7. ¡Tu sitio estará disponible en https://tu-usuario.github.io!

## ✏️ Personalizar

Para personalizar tu página, edita estos elementos en `index.html`:

- **Tu nombre**: Busca "Tu Nombre" y reemplázalo
- **Título profesional**: Busca "Desarrollador Web | Diseñador | Creativo"
- **Enlaces sociales**: Actualiza los links de GitHub, LinkedIn, Twitter
- **Email**: Cambia "tu-email@example.com"
- **Proyectos**: Edita los títulos, descripciones y enlaces
- **Habilidades**: Ajusta los porcentajes de las barras
- **Estadísticas**: Modifica los números en la sección "Sobre Mí"

## 📧 Contacto

Actualiza tu información de contacto en la sección de contacto del HTML.
